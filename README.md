# Speech Assistant

Terminal project of python that uses speech recognition and text-to-speech
This project uses the Google text-to-speech API

### Dependencies

```
pip install speechrecognition
pip install pyttsx3
pip install pyaudio
pip install playsound
pip install PyObjC
```
```
pip install PyAudio
```
### Voice Commands

- What is your name?
- What time is it
- Search
- locate
- Exit
